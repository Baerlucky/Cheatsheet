CSS Selectors & Specificity Table Reference

CSS Selectors1

Selector Meaning Example
Universal Applies to all elements in the document. *  {}
Targets all elements on the page.
Type Matches elements names. <h1>, <h2>, <p>  {}
Targets the h1, h2, p elements.
Class A period precedes the elements class attribute value. .navBar   <li>   {}
Targets an element whose class attribute has a value of navBar.
ID Pound symbol precedes theelements id attribute value. #introduction  {}
Targets an element whose id attribute has a value of introduction.
Child Matches an element that is a direct child of another. <li>><a>  {}
Targets any <a> elements that are children of the <li> element
(but not other <a> elements in the page).
Desendent Matches an element that is a desendent of another
specified element.		 <p>   <a>  {}
Targets any <a> elements that nest inside a <p> element, even
if there are other elements nested between them.
